POOL CUE VISION ASSIST — WIRING DIAGRAM

AIPI 590 Final Project · Raspberry Pi 4
★ NEW: Pi Camera Module (CSI) ★ NEW: USB Speaker ★ NEW: Laser Pointer (on cue tip) ↩ REUSED: MPU6050 · LCD · LEDs · Button · Buzzer · HC-SR04
New components require zero GPIO wiring: Pi Camera → CSI ribbon port (lift latch, insert ribbon, close latch)  |  USB Speaker → any USB-A port  |  Laser pointer → taped/clipped to cue tip (no electrical connection to Pi)
★ How to connect the new components:

Pi Camera Module: Lift the plastic latch on the CSI port (near HDMI ports). Insert ribbon cable with silver contacts facing the HDMI side. Push latch down to lock. No software setup needed on Pi OS Bullseye+.

USB Speaker: Plug USB-A cable into any of the Pi's USB ports. Set as default audio output once: sudo raspi-config → System Options → Audio → USB or amixer cset numid=3 2

Laser pointer: Tape or rubber-band it to the cue tip. Turn it on before use. The Pi camera detects the dot optically — no wiring to the Pi.
